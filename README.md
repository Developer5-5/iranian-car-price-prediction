# Iranian Car Price Prediction Using Machine Learning

This project implements a Machine Learning Regression model to predict the prices of popular Iranian cars (such as Pride, Peugeot 206, Samand, and Dena) based on features like mileage, vehicle age, and body condition.

## Technologies Used
- **Pandas & NumPy:** For data manipulation and feature engineering.
- **Matplotlib & Seaborn:** For exploratory data analysis (EDA) and visualizations.
- **Scikit-learn:** For data splitting, model training, and performance evaluation.

## Project Structure
- `car_prediction.py`: The main Python script containing the entire pipeline from dataset creation to evaluation and inference.
